The Green Domains dataset

As a small and independent non-profit organisation, we've been maintaining the world's largest open dataset on green hosting providers, the Green Domains dataset, since 2012. Every day, across the globe, aroundÂ seven million checks against it are made using our Green Web Check API and Green Web Check tools.

How can my organisation be included in the dataset?

Hosting providers can submit a verification request to us to demonstrate they are green. This process is free of charge.

Once verified, hosting providers are included in the Green Domains dataset by default and for free.

Verified hosting providers can choose to further support our work by paying a modest fee to be included in our Green Hosting Directory service.

Whether you want to be included in just the dataset or the directory as well, the starting point is the same - submit a verification request.
